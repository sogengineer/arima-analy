# スコア計算

指定したレースの出走馬に対して7要素スコアリングを実行します。

## 引数

- `$ARGUMENTS`: レースIDまたはレース名（部分一致）

## スコア配分

| 要素 | 重み |
|------|------|
| 直近成績 | 25% |
| コース適性 | 18% |
| 距離適性 | 15% |
| 上がり3F能力 | 7% |
| G1実績 | 5% |
| ローテ適性 | 15% |
| 騎手能力 | 15% |

## 実行コマンド

```bash
npx tsx src/index.ts score --race "$ARGUMENTS"
```

## 使用例

```
/スコア計算 1
/スコア計算 有馬
```

## 出力

- 総合スコアランキング（馬番順位、各要素スコア）
- 上位5頭の詳細分析（強み・課題）
- スコア分布（有力候補/注目馬/一般馬/厳しい）

## 事前確認

レースIDを確認するには `/レース一覧` を実行してください。
