# 有馬記念分析システム ヘルプ

利用可能なコマンド一覧を表示します。

## データ管理コマンド

| コマンド | 説明 |
|----------|------|
| `/データ取得 <URL>` | JRA URLからデータ取得・抽出 |
| `/arima-import-url <ファイル>` | JSONをDBにインポート |

## 一覧表示コマンド

| コマンド | 説明 |
|----------|------|
| `/レース一覧` | 登録済みレース一覧 |
| `/馬一覧` | 登録済み馬一覧（血統付き） |
| `/血統分析` | 種牡馬統計 |

## 分析コマンド

| コマンド | 説明 |
|----------|------|
| `/スコア計算 <レースID>` | 6要素スコアリング |
| `/コース分析` | 中山2500m適性分析 |

## スコア配分

```
総合スコア =
  直近成績 × 25% +
  コース適性 × 20% +
  距離適性 × 15% +
  上がり3F × 15% +
  G1実績 × 15% +
  ローテ × 10%
```

## 基本ワークフロー

1. `/データ取得 <JRA URL>` でデータ抽出
2. `/arima-import-url <JSON>` でDBインポート
3. `/レース一覧` でレースID確認
4. `/スコア計算 <ID>` でスコアリング
