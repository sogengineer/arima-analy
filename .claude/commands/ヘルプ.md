# 有馬記念分析システム ヘルプ

利用可能なコマンド一覧を表示します。

## データ管理コマンド

| コマンド | 説明 |
|----------|------|
| `/データ取得 <URL>` | JRA URLからデータ取得・抽出 |
| `/arima-import-url <ファイル>` | JSONをDBにインポート |

## 一覧表示コマンド

| コマンド | 説明 |
|----------|------|
| `/レース一覧` | 登録済みレース一覧 |
| `/馬一覧` | 登録済み馬一覧（血統付き） |
| `/血統分析` | 種牡馬統計 |

## 分析コマンド

| コマンド | 説明 |
|----------|------|
| `/スコア計算 <レースID>` | 7要素スコアリング |
| `/コース分析 [会場]` | 会場別コース適性分析（省略時は全会場） |

## スコア配分

```
総合スコア =
  直近成績 × 25% +
  コース適性 × 18% +
  距離適性 × 15% +
  上がり3F × 7% +
  G1実績 × 5% +
  ローテ × 15% +
  騎手能力 × 15%
```

## 基本ワークフロー

1. `/データ取得 <JRA URL>` でデータ抽出
2. `/arima-import-url <JSON>` でDBインポート
3. `/レース一覧` でレースID確認
4. `/スコア計算 <ID>` でスコアリング
